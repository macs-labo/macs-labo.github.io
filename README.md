# 利用規約とデータ許諾に関する重要事項

## 主旨
本プロジェクトの関連アプリ（データベース自動更新スクリプト、ACFinderBE 等）及び農薬情報データベース（MacsDB）は、日本の農薬情報を誰もが SQLite 形式で利用できる「公共インフラ」として維持することを目的としています。

このため、本プロジェクトのソースコード及びデータベースは MIT License で公開されていますが、継続的な無料データ配信を維持するため、商用・非商用を問わず利用者は以下の **「メンテナンスへの協力」** に同意したものとみなします。
「本家が最新であり続けること」が、あなたのシステムの安定に直結します。共存共栄の精神でメンテナンスへの参加をお願いします。

## 商用・有料サイトでの利用について
本データベースおよび ACFinderBE を使用した有料サービスの構築、自社システムへの取り込み等は完全に自由です。利用料も報告義務もありません。

**重要：データ利用に関する許諾について**
`toroku.csv` および `acis.db` は FAMIC（農林水産消費安全技術センター）の公開データを元に作成しています。
これらを利用して不特定多数にサービスを提供する場合は、有償・無償に関わらず、**利用者自身で FAMIC の許諾を得る必要があります。**

## メンテナンスへの協力
1. **不具合の報告**: データの不備を発見した際は、Issue または[農薬検索研究会fb](https://www.facebook.com/groups/272174746319343)で速やかに共有してください。
2. **データ改訂等への対応**: FAMIC データの形式変更や、RACコード等の改訂に気が付いた場合は 1 と同様に共有するか、可能であれば本リポジトリへ修正コードを提供（Pull Request）してください。特に、RAC コードは加除変更点のみの情報をいただけると、大変助かります。
3. **積極的な情報提供**: FAMIC の農薬登録情報や各種毒性情報と RAC コードだけでは、農薬のラベル情報にも及びません。ラベル記載事項だけでも良いので、水産動植物への影響以外の使用上の注意事項をご存じの方は、1 と同様に共有いただければ、各検索システムで表示できるようにします。農薬メーカー様におかれましては、FAMIC の「水産動植物への影響」のような形で Excel にまとめたデータをご提供いただけると幸甚です。
4. **公開DBの維持**: 本リポジトリの GitHub Actions が正常に動作し、GitHub Release で常に最新の MacsDB が無料公開されている状態を、利用者全員で守ってください。
## 関連プロジェクト・リソース

本プロジェクトを構成する各リポジトリおよび公開サイトのリンクです。

* **[MacsDB (SQLite/CSV)](./data/)** 本リポジトリ内で生成・配布されている農薬情報データベースの本体です。
* **[自動更新スクリプト](./cron/)** FAMIC データを自動取得し、MacsDB を生成する心臓部。GitHub Actions で定期実行されます。
* **[ACFinderBE](https://github.com/macs-labo/acfinder)** ブラウザ版 ACFinder の公開リポジトリです。
* **[携帯農薬検索実験室別室](https://macs-labo.github.io/)** 本プロジェクトのサーバーレス運用デモです(構築中)。
* **[携帯農薬検索実験室本室](https://macs.xii.jp/)** 最新の動的機能が稼働している開発・運用メインサイトです。
